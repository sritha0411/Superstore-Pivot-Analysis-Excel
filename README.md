# Superstore-Pivot-Analysis-Excel

## Dataset
Superstore sales dataset containing order, customer, product, region, and sales information.

## Tools Used
- Microsoft Excel

## Analysis Performed
- Total sales by category using pivot tables
- Sales analysis by region and segment
- Interactive slicers for filtering
- Pivot charts for sales visualization

## Limitations
- Profit column was not available, so profit margin analysis was skipped.

## Key Insights
- Consumer segment contributes the highest sales
- West region is the top-performing region
- Technology and Furniture categories lead in sales
